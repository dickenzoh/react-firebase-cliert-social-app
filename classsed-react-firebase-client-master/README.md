To run this on your localhost follow these instructions:

## 1: API Base URL

Set this as ur proxy value in package.json: https://europe-west1-socialape-d081e.cloudfunctions.net/api

## 2: Install packages

run `npm install`

## 3: Run project

run `npm start`

## 4: Open it

go to [http://localhost:3000]
